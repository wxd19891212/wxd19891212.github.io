在 SDImageCache.h 中你可以看到关于 SDImageCache 的描述:

SDImageCache maintains a memory cache and an optional disk cache.

SDImageCache包括内存缓存和磁盘缓存，内存缓存使用的是继承自 NSCache 的AutoPurgeCache，而磁盘缓存就是基于文件的读写。
